ICD9 Predictor from Medical Notes (TDI Capstone Project)

This is app will use take medical notes (discharge summary, office visit note...) and output a list of potential ICD9 codes.

Medical Note:

The app uses a machine learning model trained on MIMIC-III data set. The Natual Language Processing model trained in this app is comprised of two parts: a Gradient Boosted Machine model using n-grams of ICD9 code description learning the log loss. Then a two layer Neural Network learning the residuals using internal word embedding.
